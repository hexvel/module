# Справочник команд бота
## Важное оглавление
Вам нужно запомнить следующие данные:
- {} и [] - фигурные скобки обозначают обязательные аргументы, а квадратные, соответственно, необязательные

> Префиксы делятся на несколько групп:
> это префикс команд, префикс скриптов, префикс админ а также префикс доверенных.

> По умолчанию префиксы имеют следующие значения:
> > команды - х, скрипты - .с, админ - .а, доверенных #

## Команды для работы с друзьями

| Команда | Описание |
| --- | --- |
| **х +др {пользователь}** | Добавление пользователя в друзья |
| **х -др {пользователь}** | Удаление пользователя из друзей |
| **х :др** | Отколняет все заявки из друзей и оставляет их в подписчиках |

## Команды для настройки префиксов

| Команда | Описание |
| --- | --- |
| **х !команда {префикс}** | изменяет префикс команд |
| **х !скрипт {префикс}** | изменяет префикс скриптов |
| **х !админ {префикс}** | изменяет префикс администратора |

## Команды для работы со скриптами

| Команда | Описание |
| --- | --- |
| **с автообл** | включает или выключает скрипт автообложенки |
| **с обложка {тема}** | меняет тему (всего 3, по умолчанию 1) |
| **с автоферма** | включает или выключает скрипт автофермы |
| **с +статус** | включает обновление автостатуса |
| **с -статус** | выключает обновление автостатуса |
| **с статус {текст}** | создаёт шаблон для автостатуса |
| **с автоофл** | включает или выключает скрипт автоофлайна |
| **с автоонл** | включает или выключает скрипт автоонлайна |
